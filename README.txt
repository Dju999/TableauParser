Скрипт со состоит из нескольких частей частей 
0) Выгрести исходные файлы отчётов с сервера Tableau по API
1) Исходные файлы отчётов => XML документы (удаляем экстракты)
2) XML => json

1. Запуститить prepare_data.sh:
	bash .prepare_data.sh

Скрипт создаёт директорию "./files" и все поддиректории - сюда там будут входные и выходные файлы

Запускается get_workbooks.py. Происходит подключение к серверу Tableau (логин и пароль внутри скрипта). В текущей директории files созаётся папка INPUT, куда выкачиваются все доступные workbook.
Затем происходит предобработка загруженных файлов:
	- файлики .twb копируем из input в output изменений
	- файлики .twbx распаковываются и удаляется папка с экстрактом
Файлы, созданные в папке input - обычные XML, которые можно подавать на вход питоновскому скрипту

3. Генерация json 

В качестве аргумента комендной строки скрипт parse_twb.py получает имя .twb файла
В stdout выводится сформированный json
	пример использования:
	>> python parse_twb.py example.twb > example.json

все json сохраняем в ./output_json

генерируем log.txt с содержимым json
